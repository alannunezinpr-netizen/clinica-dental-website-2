# Clínica Dental & Dentista Pediátrico - Website

Website premium para clínica dental con enfoque pediátrico.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Instalación y ejecución

```bash
# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev

# Abrir http://localhost:3000
```

## Build para producción

```bash
npm run build
npm start
```

## Características

- ✅ Mobile-first responsive
- ✅ Logo original SVG inline
- ✅ Horario dinámico ("Hoy: ...")
- ✅ CTAs optimizados (Llamar + Cómo llegar)
- ✅ Sticky bar móvil
- ✅ Glassmorphism cards
- ✅ Smooth scroll con reduced-motion
- ✅ Accesibilidad (focus rings, contraste)
- ✅ Performance optimizado
